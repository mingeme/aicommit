# aicommit

`aicommit` is an AI-powered git commit message generator.

## Installation

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Link the CLI tool globally
npm link
```

## Development

```bash
# Run in development mode
npm run dev

# Build the project
npm run build

# Run tests
npm test
```

## Usage

```bash
# Show help
aicommit --help

# Run
aicommit
```
