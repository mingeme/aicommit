# aicommit

`aicommit` is an AI-powered git commit message generator.

## Installation

```bash
# Install from repo source
pnpm install mingeme/aicommit

# Install dependencies
pnpm install

# Build the project
pnpm run build

# Link the CLI tool globally
pnpm link
```

## Development

```bash
# Run in development mode
pnpm run dev

# Build the project
pnpm run build

# Run tests
pnpm test
```

## Usage

```bash
# Show help
aicommit --help

# Run
aicommit
```
